# LibraryProject

This is a basic Django project created for the Introduction to Django module in the Alx_DjangoLearnLab.

## Steps

1. Installed Django
2. Created project: `django-admin startproject LibraryProject`
3. Ran server: `python manage.py runserver`

## Project Structure

- `manage.py`
- `settings.py`
- `urls.py`
